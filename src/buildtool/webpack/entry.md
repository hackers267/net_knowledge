# 入口配置

webpack中的入口配置主要分为[单入口配置](#单入口配置)和[多入口配置](#多入口配置);

## 单入口配置

单入口配置是入口配置中最简单的配置，只需要给webpack配置中的entry字段配置一个入口文件的字符串即可。如下：

```js
module.exports = {
  // 单入口配置
  entry: './src/main.js',
  // 其它必要配置
}
```

## 多入口配置

多入口配置不同于单入口配置，多入口配置是一个object对象，由入口名称和入口文件的地址组成。如下：

```js
module.exports = {
  // 多入口配置
  // entry字段是一个object对象，其中的键是入口名称，而值为入口文件的文件路径
  entry: {
    main: './src/main.js',
    app: './src/app.js'
  },
  // 其它必要配置
}
```
